129,看见题目人先傻掉了，英语能力有待提高，然后是理解题意后发现应该是简单题，加之前无重复，加之后重复，那重复的一个子集中一定包括末尾。
进一步分析，如果因为加入最后一个产生重复，那么最差的情况下重复的每个子集之和为全部集合，即各占一半。那么最多要查K/2次确定1个字母能否放入
最差需要判断L次才能知道下一个字母，乘法原理，LK^2次计算可得解，无论如何都不可能超时！
